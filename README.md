# Stepminia

Desktop finger stepmania dance-pad.

Assebly instructions [here](ASSEMBLY.md).

## PCB

Made in Kicad. Find the production files in the pcb folder to have it manufacured (JLCPCB).

You may notice the simple and integrated pcbs. This is because I wanted to learn how to integrate a microcontroller (rp2040) into my pcb design.. but when I went to have it made it was gonna cost $100! Its not that much but I have other things I wanna buy right now. So I made the simple version that just has holes for a Waveshare Zero to be soldered on.
